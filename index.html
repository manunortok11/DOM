DOM e formulários

DOM

DOM - Data Object Model é uma API(Aplication programing interface) que visa simplificar a forma de acessar a estrutura de um documento HTML. Facilitando criar, modificar, adicionar, retirar e manipular elementos do HTML

Formulários

São elementos HTMl que possibilita a interação do usuário com a página web. Como exemplo temos caixas de texto, botões, caixas de seleção dentre outros. As tags form, label, input, sao tags de elementos de formularios.
